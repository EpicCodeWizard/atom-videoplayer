
# Videoplayer Package

Playback video files.

[![Build Status](https://travis-ci.org/tong/atom-videoplayer.svg?branch=master)](https://travis-ci.org/tong/atom-videoplayer)

### Supported formats
* 3gp
* avi
* mov
* mp4
* m4v
* mkv
* ogv
* ogm
* webm
